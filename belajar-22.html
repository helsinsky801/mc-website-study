Pengantar Ichimoku Cloud

Ichimoku Cloud adalah sistem analisis teknikal komprehensif yang dikembangkan oleh Goichi Hosoda pada tahun 1930-an di Jepang. Sistem ini dikenal sebagai "Ichimoku Kinko Hyo" yang berarti "chart balance at a glance". Ichimoku Cloud menyediakan semua informasi yang diperlukan trader dalam satu pandangan: support/resistance, momentum, trend direction, dan entry/exit signals.

Berbeda dengan indikator Barat yang fokus pada satu aspek, Ichimoku Cloud mengintegrasikan multiple timeframe analysis, moving averages, dan momentum indicators menjadi satu sistem yang powerful. Cloud (awan) yang menjadi komponen utama memberikan visual yang jelas tentang kondisi pasar saat ini.

📊

Komponen Ichimoku Cloud

🔍 Lima Komponen Utama

Ichimoku terdiri dari 5 garis yang masing-masing memiliki fungsi spesifik.

  • Tenkan-sen (Conversion Line)
    (High + Low) / 2 untuk 9 periode - Moving average jangka pendek
  • Kijun-sen (Base Line)
    (High + Low) / 2 untuk 26 periode - Moving average jangka menengah
  • Senkou Span A (Leading Span A)
    (Tenkan-sen + Kijun-sen) / 2, diproyeksikan 26 periode ke depan
  • Senkou Span B (Leading Span B)
    (High + Low) / 2 untuk 52 periode, diproyeksikan 26 periode ke depan
  • Chikou Span (Lagging Span)
    Harga penutupan saat ini, diproyeksikan 26 periode ke belakang
📖

Cara Membaca Ichimoku

  • The Cloud (Kumo)
    Area antara Senkou Span A dan B - support/resistance dinamis
  • Cloud Color
    Hijau = bullish, merah = bearish, tipis = netral
  • Price vs Cloud
    Di atas cloud = uptrend, di bawah cloud = downtrend, di dalam cloud = sideways
  • Lagging Span
    Di atas harga = bullish, di bawah harga = bearish
  • TK Cross KJ
    Tenkan cross Kijun - sinyal momentum jangka pendek
📈

Sinyal Ichimoku

⚡ Sinyal Trading

Ichimoku memberikan berbagai sinyal berdasarkan interaksi komponen-komponennya.

  • Cloud Breakout
    Harga break di atas cloud = buy signal, break di bawah cloud = sell signal
  • TK Cross KJ
    Tenkan cross Kijun ke atas = bullish, ke bawah = bearish
  • Chikou Confirmation
    Chikou di atas harga masa lalu = konfirmasi bullish
  • Cloud Twist
    Senkou A cross Senkou B - perubahan momentum jangka panjang
  • Kumo Breakout
    Harga break cloud dengan volume tinggi = sinyal kuat
🎯

Trading dengan Ichimoku

  • Trend Following
    Trade sesuai arah cloud - di atas cloud untuk long, di bawah cloud untuk short
  • Cloud Bounce
    Entry pada bounce dari cloud edge sebagai support/resistance
  • Multiple Signal Confirmation
    Tunggu konfirmasi dari TK cross KJ dan Chikou sebelum entry
  • Timeframe Analysis
    Gunakan timeframe mingguan untuk trend utama, harian untuk entry
  • Risk Management
    Stop loss di cloud edge terdekat, target profit di cloud edge berikutnya
☁️

Analisis Cloud

🌥️ Interpretasi Cloud

Cloud adalah komponen terpenting Ichimoku yang memberikan gambaran kondisi pasar.

  • Cloud Thickness
    Cloud tebal = support/resistance kuat, cloud tipis = level lemah
  • Cloud Color
    Hijau = demand > supply, merah = supply > demand
  • Future Cloud
    Cloud yang diproyeksikan ke depan membantu antisipasi level kunci
  • Cloud as Target
    Gunakan cloud sebagai target profit atau stop loss
  • Cloud Break Confirmation
    Break cloud harus ditutup di luar cloud untuk konfirmasi
